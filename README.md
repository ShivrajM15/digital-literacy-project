## Project Overview
This repository contains my Digital Literacy Project completed as part of the CSE0001 course. 

As a Student Digital Ambassador, this project focuses on building essential digital skills such as creating a professional online presence, using digital tools effectively, practicing safe internet behavior, and understanding cybercrime awareness.

The project is divided into five tasks, each aligned with a specific module of the course. All deliverables, including screenshots, write-ups, and resources, are organized into their respective folders.
